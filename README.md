# Student Management System (Java)

## About Project
This is a simple console-based Java project.

## Features
- Add Student  
- View Students  
- Simple Menu System  

## Tech Used
- Java  
- ArrayList  
- Scanner  

## How to Run
1. Compile: `javac StudentManagementSystem.java`  
2. Run: `java StudentManagementSystem`
